## Redesign страницы «Дауыс» (Triage Voice Chat)

Полностью уходим от «двух прямоугольников на весь экран». Чат превращается в кинематографичное пространство разговора с живым голосовым визуализатором, атмосферным aurora-фоном и плавающими сообщениями-капсулами. Микс из всех трёх предложенных направлений.

### Что меняется на экране

```text
┌──────────────────────────────────────────────────────────────┐
│  eyebrow · title · description                (LIVE · lang)  │
├──────────────────────────────────────────────────────────────┤
│ ╭──────────────────╮   ╭────────────────────────────────╮    │
│ │                  │   │  ░░ aurora glow behind ░░       │   │
│ │   VOICE ORB      │   │    ● SauBol · 12:41            │    │
│ │   мятная сфера   │   │   ╭─ капсула ai ──╮             │   │
│ │   с волной       │   │   ╰────────────────╯            │   │

│ │                  │   │              ╰────────╯         │   │

```

### Ключевые визуальные приёмы

1. **Aurora Cocoon фон чата** — внутри карточки чата медленно движется мятно-фиолетовый градиент (используем существующие `aurora`/`--mint` токены + `radial-gradient` слои с низкой непрозрачностью, `blur(60px)`). Никаких hex-цветов в компонентах.
2. **Editorial Split (асимметрия 320px / 1fr)** — слева узкая колонка «сессия», справа поток диалога. Убираем правый сайдбар в его текущем виде; критический риск 103 и «Detected symptoms» переезжают в левую колонку компактными блоками под орбом.
3. **Voice Orb** — круглая мятная сфера ~220px с концентрическими пульс-кольцами (SVG + CSS-анимация), в центре таймер сессии и мини-эквалайзер из 5 баров (`spark-bar`, уже есть в `styles.css`). Когда включён `dictating` — кольца ускоряются и появляется внешнее halo-свечение.
4. **Floating Bubbles** — сообщения больше не одинаковой ширины во всю строку:
   - AI: слева, `max-w-[68%]`, полупрозрачный `bg-card/60` с `backdrop-blur`, тонкий градиентный бордер (`border-white/8` + inset ring), скруглены `rounded-[22px]` с одним «хвостиком-скосом» у аватара.
   - Пациент: справа, `max-w-[62%]`, чуть уже, фон `bg-[color:var(--mint-soft)]` с мятным ободком.
   - Каждое сообщение появляется с `animate-fade-in`.
   - Аватары — круги 32px с мягким halo (мятный для AI, нейтральный для пациента).
5. **Плавающий composer** — прикреплён к нижней части области чата (`sticky bottom-3`), в виде «пилюли» с backdrop-blur, тенью и внутренним свечением. Слева крупная круглая mic-кнопка 48px с halo-анимацией во время записи (двойное пульс-кольцо). Справа отправка в виде круглой кнопки со стрелкой.
6. **Quick-chips** остаются, но становятся glass-таблетками с тонкой линией и hover-lift (`hover-scale`).
7. **Индикатор «thinking»** — вместо трёх точек рендерим shimmer-строку «SauBol ойлануда…» с мятным bar-loader'ом под ней.

### Технические детали

- Файл: только `src/routes/triage-voice.tsx`. Логика (`send`, `toggleDictation`, `triageChat`, i18n, quick-chips, transcript) не меняется — переписываем только разметку и классы.
- Новые утилиты по необходимости добавляем в `src/styles.css` через `@utility` (v4-стиль): `@utility orb-pulse`, `@utility halo-ring`, `@utility chat-aurora`. Никаких hardcoded цветов — только `var(--mint)`, `var(--mint-soft)`, `var(--card)`, `var(--border)`, `var(--foreground)`.
- Для «плавающих капсул» используем уже существующие семантические токены + `backdrop-blur-xl` (Tailwind-утилита, без ручных `-webkit-` префиксов).
- Grid переключается с `lg:grid-cols-[1fr_360px]` на `lg:grid-cols-[300px_1fr]`; сайдбар 103 + Detected Symptoms уплотняется в компактные карточки под орбом.
- Auto-scroll, streaming-статус, LIVE-бейдж, копирование транскрипта, переключение языков — сохраняем как есть.
- Никаких изменений бэкенда, `triage-ai.functions.ts`, роутинга, i18n-словарей.

### Проверка после реализации

- Визуально: чат больше не выглядит как две плитки; орб анимируется, сообщения-капсулы разной ширины, composer «плавает».
- Функционально: отправка текста, диктовка (Chrome/Edge), quick-chips, смена языка, копирование транскрипта, thinking-state — всё работает.
- Тёмная тема (единственная в проекте) читаема, контраст текста в капсулах ≥ AA.
